    Building RetroTxt

    Windows

    Tasks do not run on PowerShell, Windows users should use Windows Subsystem for Linux.

    Clone, initialize and branch the repo

    gh repo clone bengarrett/RetroTxt
cd RetroTxt
task init

    # Create a dev branch
git checkout -b dev

    # Create a pull request to merge the dev branch into main
gh pr create
gh pr status
gh pr merge

    Package dependencies to newer versions

    yarn outdated
yarn upgrade-interactive --latest

    Update VERSION stamp

    edit Taskfile.yml

    vars:
  VERSION: "0.0.0"

    # apply VERSION stamp to the manifest and package json files.
task version-set
# or set the VERSION and then submit to GitHub
task commit
