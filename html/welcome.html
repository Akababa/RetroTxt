Welcome to RetroTxt, the extension that turn pieces of ANSI text art and ASCII, NFO plain text into HTML.

To enable the full functionality of RetroTxt you need to select some Apply RetroTxt to checkboxes in the Config Options. Each setting is optional but will require additional browser permissions.

Example

Below is an example of our ANSI logo rendered as HTML that includes an information header. Each header item has a mouse-over triggered short description. The bold items are links that cycle through options.
